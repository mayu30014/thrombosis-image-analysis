# thrombosis-image-analysis pipeline

This repository contains custom semi-automated Python pipelines for analyzing thrombosis and cell density using `napari` and `scikit-image`.

## Requirements
- Python 3.13+
- napari 
- scikit-image
- opencv-python
- scikit-learn
- pandas
- tifffile
- openpyxl
